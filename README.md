# HiChIP-pipeline

This is a pipeline for HiChIP analysis. 

## Data information

1. ChIP target: H3K9me3.

2. Groups: KO (2 replicates) and WT (2 replicates).

3. Cell line: PANC-1 (PDAC cell model).

## Requirements

### For ChIP-seq

1. samtools

2. deepTools

3. macs3

### For Hi-C

1. HiC-Pro

2. juicebox

3. Cooltools

4. FANC

### For HiChIP

1. FitHiChIP

2. HiCCUPS
